## 其他物品獲得方式
- 玩家頭顱(player_hend)
  1. 獲取一本"書和羽毛筆"
  2. 在書中寫下目標玩家的ID
  3. 點擊完成
  4. 利用鐵砧將"書和羽毛筆"物品的名字改為"pd"(player hend)
  5. 將此物品丟到地上
  6. 書本將會成為玩家頭顱
